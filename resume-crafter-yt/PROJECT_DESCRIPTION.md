# Resume Crafter - Brief Description

## Overview

Resume Crafter is an AI-powered web application that generates professional, personalized resumes based on user input and career stage. Users can create accounts, select their profile type (Student, Intern, or Professional), and generate customized resume templates using OpenAI's GPT-4o-mini model. The platform includes resume editing capabilities and PDF export functionality.

## Technologies Used

**Backend:**
- Python 3.12
- Flask 3.1.1 (Web Framework)
- Flask-SQLAlchemy 3.1.1 (Database ORM)
- Flask-Bcrypt 1.0.1 (Password Security)
- SQLite (Database)
- OpenAI API 1.93.2 (AI Resume Generation)

**Frontend:**
- HTML5
- CSS3
- JavaScript
- Jinja2 Templates

**Key Features:**
- User authentication and session management
- Category-specific resume generation (Student/Intern/Professional)
- Real-time resume editing
- PDF export functionality
- Input validation and error handling
